JavaScript

  • JavaScript is a high-level, interpreted programming language.
  • Alongside HTML and CSS, JavaScript is one of the three core technologies of the World Wide Web.
  • JavaScript enables interactive web pages and thus is an essential part of web applications.
  • Most JavaScript programs contain many JavaScript statements. Semicolons separate JavaScript statements.

Some common Javascript terminologies:

Variable
JavaScript variables are containers for storing data values.The data can either be an integer, boolean or a string value. Those data can later be accessed and retrieved when needed. A beautiful functionality of a variable is, it gives opportunity for a user to modify the value of any specific state of the program. All variables must be indentified with unique names. Identifiers are case-sensitive in javascript. In javaScript '=' operator is an assignment operator, not an equal to operator. Read more

Conditionals
Conditionals control behavior in JavaScript and determine whether or not pieces of code can run.
if
The most common type of conditional is the if statement, which only runs if the condition enclosed in parentheses () is truthy.
else
An if statement can be extended with an else statement, which adds another block to run when the if conditional doesn’t pass.
if else
An if statement can also be extended with an else if statement, which adds another conditional with its own block.
Though multiple if else conditionals can be used, but note that only the first else if block runs when it finds the statement to be true. JavaScript skips any remaining conditionals after it runs the first one that passes. Read more!

Function
A JavaScript function is a "recipe" of instructions whose purpose is to accomplish a well-defined task. In other words, Functions are blocks of code that can be named and reused. When the JavaScript executes a function, it processes these instructions, one by one, until there are no more instructions in the function to execute.
Some functions can be thought of as "processing units" that accept data from one end and produce results at the other. A function's input is called its parameters; its output is called the return value.
For example, a JavaScript function whose purpose is to compute the square of a number might be defined as follows:

function square (number)
{
var result = (number*number);
return result;}

Loops
Loops offer a quick and easy way to do something repeatedly. There are many different kinds of loops, but they all essentially do the same thing: they repeat an action some number of times. The various loop mechanisms offer different ways to determine the start and end points of the loop. There are various situations that are more easily served by one type of loop over the others. A loop can be compared as a computerized version of the game where you tell someone to take X steps in one direction then Y steps in another; for example, the idea "Go five steps to the east" could be expressed this way as a loop:
var step; for (step = 0; step < 5; step++) {
// Runs 5 times, with values of step 0 through 4.
console.log('Walking east one step');
}

Arrays
Arrays are container-like values that can hold other values. The values inside an array are called elements.
For example, var breakfast = ["coffee", "croissant"]; where breakfast is an array which holds two value "coffee" and "croissant" that can be used in later events.
Array elements don’t all have to be the same type of value. Elements can be any kind of JavaScript value — even other arrays.To access one of the elements inside an array, one’ll need to use the brackets and a number like this: breakfast[1]. JavaScript arrays begin at 0, so the result will be "croissant".

Objects
JavaScript is an object-oriented programming language.To develop applications in JavaScript effectively you need to understand how to work with objects. Objects can encapsulate both data (several different data items in a single object) and behavior. The behavior of an object is defined by defining one or more methods of the object. In JavaScript, a method is a JavaScript function that is invoked through an object. literals are the fixed values in the program code. The following table illustrates different literals used in object programming in Javascript.
