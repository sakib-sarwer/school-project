Programming

Programming is the process of taking an algorithm and encoding it into a notation, a programming language, so that it can be executed by a computer. In general, computer programming can be seen as a process of designing, writing, testing, troubleshooting and debugging, documenting and maintaining any computer programs.

In general, the programming process can be seen as a problem solving process where the programmer should do the following:

  1. Understand the problem: Focus on the problem. Analyze the problem. Refine the problem by decomposing it into sub-problems.
  2. Devise a plan : Design the program.
  3. Carry out the plan : Write the program code in a programming language
  4. Evaluate the results : Test the program for errors and quality/usability etc.

Programming language

Programming languages are artificial languages used to write instructions that can be either directly interpreted and executed by the computer or compiled into machine-readable instructions and then executed by a computer.
  • Set of formal specifications concerning vocabulary, syntax, and semantics (meaning).
  • Used to describe the process and data structures.
